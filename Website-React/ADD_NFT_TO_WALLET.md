# 💼 如何将 NFT 添加到钱包

## 📋 步骤说明

### 方法 1：使用"添加到钱包"按钮（推荐）

1. **铸造 NFT**
   - 在商品列表中找到你想要铸造的商品
   - 点击"铸造 NFT"按钮
   - 在钱包中确认交易
   - 等待交易确认

2. **添加到钱包**
   - 铸造成功后，会出现"💼 添加到钱包"按钮
   - 点击该按钮
   - 如果网络不匹配，系统会提示你切换网络
   - 在钱包中确认添加请求

3. **查看 NFT**
   - 打开 MetaMask 钱包
   - 切换到"NFT"标签页
   - 你应该能看到你的 NFT

### 方法 2：手动添加（如果自动添加失败）

#### 在 MetaMask 中手动添加：

1. **打开 MetaMask**
   - 点击钱包扩展图标
   - 确保连接到正确的网络（Tenderly 测试网）

2. **添加 NFT**
   - 点击"NFT"标签页
   - 点击"导入 NFT"
   - 输入以下信息：
     - **合约地址**：你的 NFT 合约地址（从部署页面获取）
     - **Token ID**：你的 NFT Token ID（铸造成功后显示）

3. **确认添加**
   - 点击"添加"
   - NFT 应该会出现在你的钱包中

## ⚠️ 常见问题

### 问题 1：无法添加 NFT

**可能原因：**
- 网络不匹配（合约在 Tenderly 测试网，钱包在其他网络）
- Token ID 不正确
- 合约地址不正确

**解决方法：**
1. 确保钱包连接到 Tenderly 测试网（Chain ID: 623352640）
2. 检查合约地址是否正确（42 个字符，以 0x 开头）
3. 确认 Token ID 是否正确（铸造成功后显示的 ID）

### 问题 2：找不到"添加到钱包"按钮

**可能原因：**
- NFT 尚未铸造
- 交易尚未确认

**解决方法：**
1. 先点击"铸造 NFT"按钮
2. 等待交易确认（可能需要几分钟）
3. 确认成功后，按钮会自动出现

### 问题 3：网络不匹配

**解决方法：**
1. 点击"添加到钱包"时，系统会提示切换网络
2. 点击"是"自动切换
3. 如果自动切换失败，手动在 MetaMask 中切换网络

## 🔧 网络配置

### Tenderly 测试网信息

- **Chain ID**: 623352640

- **区块浏览器**: https://dashboard.tenderly.co

### 在 MetaMask 中添加网络（如果需要）

1. 打开 MetaMask
2. 点击网络下拉菜单
3. 点击"添加网络"
4. 输入以下信息：
   - 网络名称：Tenderly Testnet
   - RPC URL：上面的 RPC URL
   - Chain ID：623352640
   - 货币符号：ETH
5. 点击"保存"

## 📝 获取合约地址和 Token ID

### 合约地址
- 在部署页面（`/deploy`）查看已部署的合约地址
- 或从 `.env.local` 文件中的 `VITE_NFT_CONTRACT_ADDRESS` 获取

### Token ID
- 铸造成功后，Token ID 会显示在商品详情中
- 或在区块链浏览器中查看交易详情

## 💡 提示

- 确保钱包中有足够的 ETH 支付 Gas 费用
- 铸造 NFT 需要等待交易确认（通常几秒到几分钟）
- 如果自动添加失败，可以尝试手动添加
- 某些钱包可能不支持自定义网络，需要手动添加网络配置
